# LAZYCC

## Why

This project aims to improve the programmer experience with the cpp language by adding some syntax sugar & features to the the language via a single header file

## Examples

```
var number = 10;
```
```
final pi = 3.1415;
```
```
fn<void()> say_hello = [](){
    cout << "hello" << endl;
}
```

##### Supported version: c++17 or newer
